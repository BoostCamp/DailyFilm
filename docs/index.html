AVFoundation

AVFoundation 프레임워크를 사용한 커스텀 카메라 뷰를 구현하였습니다.

카메라 전후면 전환, 포커스 모드, 플래쉬 모드 기능이 있습니다.

오픈소스를 사용하지 않고 Apple Guide를 참고하여 개발했습니다.

Core Image

애플의 이미지 프로세싱 프레임워크 Core Image를 사용하였습니다.

OpenGL ES3 api를 통한 렌더링

CIFilter를 통한 이미지 필터 적용

CIDector를 통한 얼굴 인식

SQLite-FMDB

SQLite를 활용하여 모바일 데이터베이스를 구축하였습니다.

FMDB Library를 사용하여 생산성을 높였습니다.
